Deploying Maven Artifact

Introduction

Developers note on how to deply artifact. For details see .

Details

Prerequisites

gpg key kreated and known pass phrase. See . Currently tomac.org key exists.

Steps

  1. add to .m2/settings.xml sonatype-nexus-snapshots your-jira-id your-jira-pwd sonatype-nexus-staging your-jira-id your-jira-pwd
  2. enable maven-gpg-plugin in pom.xml (default uncommented) org.apache.maven.plugins maven-gpg-plugin sign-artifacts verify sign
Deploy mvn clean deploy Give the passphrase of the key.
